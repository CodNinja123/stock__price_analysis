# Stock Price Time-Series Analysis

This repository contains my solution for the Stock Price Analysis project from roadmap.sh.

Project link: [https://roadmap.sh](https://roadmap.sh/projects/stock-price-time-series)

## Project Deliverables
* Downloaded historical data for Apple (AAPL) and Microsoft (MSFT) via `yfinance`
* Set tracking dates as the primary index
* Rendered historical closing price trends via interactive Matplotlib line charts
* Cleanly resampled structural metrics into weekly and monthly aggregation frequencies
* Analyzed stock momentum using 20-day and 200-day rolling moving averages
* Illustrated share trading activity profiles via bar volume charts

## Technologies Used
* Python 3
* Anaconda / Jupyter Notebook
* Pandas
* yfinance
* Matplotlib
